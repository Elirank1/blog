## About This Blog

This blog is built with some awesome technologies:

- **[Astro](https://astro.build)** - Ultra-fast static site generator
- **MDX** - Markdown with JSX support
- **TypeScript** - Type-safe development
- **GitHub Pages** - Free hosting

The entire source code is available on [GitHub](https://github.com/Elirank1/monorepo) under MIT license.
