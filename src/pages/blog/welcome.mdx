This is my corner of the internet. A place to think out loud, share what I'm building, and document the journey.

## What I'm Into

I like building tools that actually work. Not just proof-of-concepts, but things you can use day-to-day. Lately I've been deep into:

- **AI integrations** — finding practical ways to wire Claude and other models into real workflows
- **Automation** — if I do something twice, I want a script to do it the third time
- **Full-stack web dev** — React, TypeScript, Node.js, and whatever gets the job done

## What I'm Building Right Now

A few things are in the works:

**Suno Music Generator** — an AI-powered music creation app. Think text-to-song. It's built with React + TypeScript and it's been a fun challenge figuring out the UX for something this creative.

**Claude Automation Toolkit** — this one's inspired by OpenClaw. It connects Claude to WhatsApp, HubSpot, and Google Meet so you can automate the boring parts of your day. MCP server included.

**This blog** — built with Astro + MDX, deployed on GitHub Pages. Fast, minimal, no nonsense.
