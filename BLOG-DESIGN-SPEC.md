# Blog Design Spec — Deeplica

## Fonts
- **Headings**: MuseoModerno, Medium weight
- **Body**: Source Sans 3, Regular weight
- **Code**: JetBrains Mono

## Colors — Light Mode
- Heading text: `#967FDB` (Deeplica purple)
- Body text: `#2B4150` (dark blue-gray)
- Accent/links: `#967FDB`
- Accent hover: `#7B62C9`
- Accent light (tag chips bg): `#EDE9FE`
- Background: `#FFFFFF`
- Muted text: `#6B7280`

## Colors — Dark Mode
- Heading text: `#C4B5FD` (bright lavender)
- Body text: `#D1D5DB`
- Accent/links: `#B8A4E8`
- Accent hover: `#D0C1F0`
- Accent light (tag chips bg): `#1E1538`
- Background: `#0F0F13`
- Code block bg: `#1A1A24`
- Borders: `#1F1F2E`

## Extended Palette (Deeplica Design System)
- Deeplica Purple: `#967FDB`
- Lavender: `#B8A4E8`
- Sand: `#E8DCC8`
- Pale Blue: `#B8D4E3`
- Fog Gray: `#9CA3AF`
- Beige: `#F5F0E8`

## Gradients
- Avatar/logo: `from-violet-500 to-purple-700`
- OG image accent bar: `#967FDB → #B8A4E8`

## OG / Social Sharing
- Per-post hero image used as LinkedIn/Twitter thumbnail
- Fallback: Satori-generated OG image (dark bg, purple accent bar, Inter font)
- Branding: "Eliran Keren | Deeplica"
